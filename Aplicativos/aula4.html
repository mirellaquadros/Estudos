Formulários

<form> ... </form> indica um formulário, e tem os atributos action (indica o arquivo que vai processar os dados), method (método de requisição - GET ou POST), entre outros.

<fieldset> ... </fieldset> declara um conjunto de campos.
<legend> ... </legend> indica ao formulário uma legenda para o conjunto de campos.

<label> ... </label> indica um rótulo ao campo referenciado (por meio do atributo for com o ID do campo).

<input .../> declara um campo, podendo ser do tipo texto (type="text"), numérico (type="number"), de email (type="email"), de data (type="date"), de telefone (type="tel"), de senha (type="password"), entre outros. Pode possuir também os atributos name (nome do campo), id (identificação do campo na aplicação), value (valor do campo), placeholder (dica de preenchimento do campo), entre outros.

Botões: existem diferentes formas de criar botões, podendo ser:

<input type="submit" value="Enviar" /> indica um botão que ao clicá-lo, submete os dados do formulário para o processamento (arquivo indicado no atributo action do form). Nesse caso, o atributo value é o texto a aparecer no botão.

<input type="button" value="Enviar" /> indica um botão que ao clicá-lo, não submete os dados por padrão, sendo preciso implementar essa ação. O atributo value também é o texto a aparecer no botão.

<button type="submit">Enviar</button> também indica um botão que ao clicá-lo, submete os dados do formulário para o processamento. O texto a aparecer no botão fica declarado entre suas tags de abertura e fechamento.

<button type="button">Enviar</button> indica um botão que ao clicá-lo, não submete os dados por padrão, sendo preciso implementar essa ação. E o texto a aparecer no botão também fica declarado entre suas tags de abertura e fechamento.

Exemplo:

<h3>Dados do usuário:</h3>
<form action="" method="GET">
<fieldset>
<legend>Básico</legend>
<label for="nome">Nome: </label>
<input type="text" name="nome" id="nome" placeholder="Digite o seu primeiro nome completo" />
<br/>
<label for="email">Email:</label>
<input type="email" name="email" id="email" placeholder="Digite o seu email" />
</fieldset>
<fieldset>
<legend>Específico</legend>
<label for="dataNascimento">Data de Nascimento: </label>
<input type="date" name="dataNascimento" id="dataNascimento" /> <br/>
<label for="idade">Idade: </label>
<input type="number" name="idade" id="idade" /> <br/>
<label for="telefone">Telefone: </label>
<input type="tel" name="telefone" id="telefone" />
</fieldset>
<input type="submit" name="submeter" value="Submeter" class="botao" />
</form>

Outros tipos de campos: <textarea</textarea> indica um campo de texto livre, isto é, de múltiplas linhas. Tem os atributos: name (nome), id (identificação na aplicação), rows (quantidade de linhas), cols (colunas - largura do campo). <input type="radio" value="1" /> indica botões (radio buttons) onde somente é possível selecionar uma opção. O atributo value é o valor da seleção. <input type="checkbox" value="opcao1" /> indica botões (caixas de seleção) onde é possível selecionar uma ou várias opções. O atributo value é o valor da opção. <select> ... </select> indica uma caixa de seleção, podendo selecionar somente uma opção. As opções são declaradas através da tag <option value="10"> ... </option>, onde value é o valor da opção.
