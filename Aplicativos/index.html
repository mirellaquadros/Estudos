AULA 1

HTML

HyperText Markup Language (linguagem de marcação de hipertexto). Hipertexto se refere aos links que conectam páginas web entre si, seja dentro de um único site ou fora dele. A marcação é usada para denotar textos, imagens e outros conteúdos para exibição em um navegador web.

Cada marcação é representada por elementos (tags) como o <head>, <body>, <header>, <footer>, <img>, entre outros.

Estrutura Básica

<!DOCTYPE html> não é uma tag, mas sim uma declaração informando ao navegador o tipo/versão do html utilizada no arquivo. Neste caso, é o HTML 5.

<html> ... </html> especifica o início e fim da estrutura HTML.

<head> ... </head> é o cabeçalho da página, isto é, informações a serem lidas pelo navegador ou buscadores web. Contém:

  • Meta tags <meta> que auxiliam os mecanismos de busca a encontrarem o nosso site.
  • Título da página <title> ... </title>
  • Links <link> a arquivos externos essenciais ao site (arquivos CSS, scripts, fontes, etc.)

<body> ... </body> que contém todos os elementos do corpo da página a mostrar ao usuário

HTML - Texto

<code> ... </code> indica que o texto é um código e altera sua formatação:
<p>Em JavaScript, as constantes são declaradas usando a seguinte sintaxe:
</p> <code> const aliquota = 20.00; </code>

Saída no navegador:

Em JavaScript, as constantes são declaradas usando a seguinte sintaxe: const aliquota = 20.00;
<mark> ... </mark> destaca o texto, possuindo o efeito semelhante ao marca-texto:
<p>JavaScript é uma linguagem de programação interpretada utilizada na parte <mark>front end</mark> da aplicação web.</p>

Saída no navegador:

JavaScript é uma linguagem de programação interpretada utilizada na parte front end da aplicação web.

<blockquote> ... </blockquote> indica que o texto é uma citação aplicando um recuo para a direita:

<p>Segundo Steve Jobs ao falar com John Sculley, presidente da Pepsi antes de ser contratado pela própria Apple: </p>
<blockquote>“Algumas pessoas acham que foco significa dizer sim para a coisa em que você vai se focar. Mas não é nada disso. Significa dizer não às centenas de outras boas ideias que existem. Você precisa selecionar cuidadosamente.”</blockquote>

Saída no navegador:

Segundo Steve Jobs ao falar com John Sculley, presidente da Pepsi antes de ser contratado pela própria Apple:

“Algumas pessoas acham que foco significa dizer sim para a coisa em que você vai se focar. Mas não é nada disso. Significa dizer não às centenas de outras boas ideias que existem. Você precisa selecionar cuidadosamente.”

HTML - Imagens

<img ... /> indica que o elemento é uma imagem. Possui os atributos src (endereço da imagem/arquivo), width (largura), height (altura), title (texto acessível quando posiciona o mouse em cima da imagem), alt (texto alternativo mostrado quando não é possível carregar a imagem), entre outros.

<figure> ... </figure> usada para organizar e agrupar ilustrações, diagramas, imagens, vídeos, gráficos, entre outros elementos.

<figcaption> ... </figcaption> usada em conjunto com a tag figure, indica a legenda da ilustração em questão.

Exemplo:

<figure> <img src="estudos.jpg" alt="Estudos" width="150" height="140" /> <figcaption>Estudos Intensivos em Web</figcaption> </figure>

HTML - Audio e Vídeo

<audio> ... </audio> indica que o elemento é um áudio.

<video> ... </video> indica que o elemento é um vídeo.

<source ... /> é uma tag auxiliar de vídeo e áudio, e indica o arquivo da mídia.

Tanto a tag audio como a tag video possuem os atributos src (endereço do vídeo/arquivo), width (largura), height (altura), type (tipo do arquivo), autoplay (especifica se ao carregar a página o vídeo será executado), controls (exibe a barra de controle), entre outros.

Exemplo:

<video width="300" height="200" controls="controls">
<source src="GLG.mp4" type="video/mp4">
</video>
<br/><br/>
<audio controls="controls">
<source src="GLG.mp3" type="audio/mp3">
</audio>

HTML - Blocos

<section> ... </section> dentro do conteúdo principal, compreende uma seção da página.

<article> ... </article> indica um artigo na página - muito utilizado em blogs e sites de criação de conteúdo.

<aside> ... </aside> se refere a uma seção com outro conteúdo da página (lateral), como uma explicação, avisos, conteúdos complementares.

<div> ... </div> indica um container genérico e não tem valor semântico - é apenas uma divisão na página para fins quaisquer. Pode ser usado em qualquer local da página.
