Lista Ordenadas

<ol> ... </ol> declara uma lista ordenada.
<li> ... </li> declara um item da lista.

Exemplo:

<h4>Principais alimentos com proteína: </h4>
<ol>
<li>Carne bovina</li>
<li>Frango</li>
<li>Peixes</li>
<li>Carne suína</li>
</ol>

Saída no navegador:

Principais alimentos com proteína:

  1. Carne bovina
  2. Frango
  3. Peixes
  4. Carne suína

Listas não ordenadas

<ul> ... </ul> declara uma lista não ordenada (bastante usada para definir menus e submenus).
<li> ... </li> declara um item da lista.

type é um atributo para definir o marcador (círculo, quadrado, flecha, etc.).

Exemplo:

<h4>Listagem de bairros na região sul: </h4>
<ul>
<li>Jardim Primavera</li>
<li>Vila Mariana</li>
<li>Altos da Boa Vista</li>
<li>Residencial dos Itamaratis</li>
<li>Vila Carcará</li>
<li>Parque Americano</li>
</ul>

Saída no navegador:

Listagem de bairros na região sul:

  • Jardim Primavera
  • Vila Mariana
  • Altos da Boa Vista
  • Residencial dos Itamaratis
  • Vila Carcará
  • Parque Americano

Links

<a> ... </a> declara um hiperlink - conexão com uma outra página ou site web.

href: é usado para referenciar uma URL externa, arquivo ou local interno do site.

target: indica onde será carregada a página referenciada no link (href). Como valor, pode ser _blank (abre o endereço em nova janela), _self (normal/padrão), _parent (abre o endereço no frame pai), _top (abre o endereço em todo o corpo da janela), framename (abre o endereço no frame nomeado/especificado).

Exemplo:

<a href="https://www.tecmundo.com.br/" target="_blank">Tecmundo - Descubra e aprenda sobre tecnologia</a>

Saída no navegador:

Tecmundo - Descubra e aprenda sobre tecnologia

Iframes

<iframe> ... </iframe> permite incluir na página HTML outra página em uma seção especificada. Para isso, possui os atributos:

  • src: endereço do arquivo;

  • width: largura do iframe;

  • height: altura do iframe;

  • frameborder: espessura da borda do iframe;

  • allowfullscreen: permite o modo tela cheia;

  • allow: permite outros recursos, como o accelerometer (sensor de movimento), autoplay (execução automática), geolocation (geolocalização), microphone (controle de microfone), entre outros.

Exemplo:

<header>
<h2>Documento principal</h2>
</header>
<main>
<iframe src="https://blog.betrybe.com/html/iframe-html/" width="500"
height="200" frameborder="2">
</iframe>
</main>
