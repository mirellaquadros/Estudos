AULA 1

HTML

HyperText Markup Language (linguagem de marcação de hipertexto). Hipertexto se refere aos links que conectam páginas web entre si, seja dentro de um único site ou fora dele. A marcação é usada para denotar textos, imagens e outros conteúdos para exibição em um navegador web.

Cada marcação é representada por elementos (tags) como o <head>, <body>, <header>, <footer>, <img>, entre outros.

Estrutura Básica

<!DOCTYPE html> não é uma tag, mas sim uma declaração informando ao navegador o tipo/versão do html utilizada no arquivo. Neste caso, é o HTML 5.
<html> ... </html> especifica o início e fim da estrutura HTML.
<head> ... </head> é o cabeçalho da página, isto é, informações a serem lidas pelo navegador ou buscadores web. Contém:
● Meta tags <meta> que auxiliam os mecanismos de busca a encontrarem o nosso site.
● Título da página <title> ... </title>
● Links <link> a arquivos externos essenciais ao site (arquivos CSS, scripts, fontes, etc.)
<body> ... </body> que contém todos os elementos do corpo da página a mostrar ao usuário

HTML - Texto

<code> ... </code> indica que o texto é um código e altera sua formatação:
<p>Em JavaScript, as constantes são declaradas usando a seguinte sintaxe:
</p> <code> const aliquota = 20.00; </code>
Saída no navegador:
Em JavaScript, as constantes são declaradas usando a seguinte sintaxe: const aliquota = 20.00;
<mark> ... </mark> destaca o texto, possuindo o efeito semelhante ao marca-texto:
<p>JavaScript é uma linguagem de programação interpretada utilizada na parte <mark>front end</mark> da aplicação web.


Saída no navegador:
JavaScript é uma linguagem de programação interpretada utilizada na parte front end da aplicação web.

<blockquote> ... </blockquote> indica que o texto é uma citação aplicando um recuo para a direita: <p>Segundo Steve Jobs ao falar com John Sculley, presidente da Pepsi antes de ser contratado pela própria Apple: </p>

“Algumas pessoas acham que foco significa dizer sim para a coisa em que você vai se focar. Mas não é nada disso. Significa dizer não às centenas de outras boas ideias que existem. Você precisa selecionar cuidadosamente.”
